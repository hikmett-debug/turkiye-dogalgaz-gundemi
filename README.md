# Türkiye'nin Doğal Gaz Gündemi: Veri Görselleştirme Analizi

Bu çalışma; Türkiye'nin enerji arz güvenliği, doğal gaz ticaret hacmi ve sektörel tüketim verilerinin istatistiksel yöntemlerle incelendiği akademik bir araştırma projesidir. Analiz süreçleri, nesne yönelimli programlama prensipleriyle **RStudio** ortamında yürütülmüştür.

---
